What is StackButton?

Stackbutton is designed for DevOps and delivers an easy-to-deploy, central monitoring system for cloud-based services. It is designed to play with these technologies.

NodeJS

NodeJS - Stackbutton is built in JavaScript and runs on Node. This makes interactions from a browser quick and responsive.

Angular

Angular - Stackbutton is built in Angular and TypeScript, which allows for object-oriented web development and responsive user interactions.

OpenShift

OpenShift - is an application development and hosting platform that automates the provisioning, management, and scaling of Stackbutton alongside other containers.

PatternFly

PatternFly - Stackbutton is built in Angular and TypeScript, which allows for object-oriented web development and responsive user interactions.
